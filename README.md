# APCSP-Exam-Project
The python script for my AP Computer Science Principles AP exam.

# Description
This is a small playable game with one level, the objective is to get to the bright green finish line. Touching a red block, touching a spike, or falling off the stage restarts the game.

# Dependencies
- Turtle
- Random
- Time

# Controls
A/D - Move left and right \
W - Jump
